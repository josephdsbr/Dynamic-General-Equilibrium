# Dynamic-General-Equilibrium

A Shiny App with simple model of Dynamic General Equilibrium to see impacts of Economics Policies.

Dynamic General Equilibrium model is extensed used in Macroeconomics' researches and analysis. It's importante to understand the concepts of the model.
But, it's also important to have, specially to graduate students, some tools to visualize and get some tricks of this model.

Because of it, we created a Visualize Application to help the user understand the impacts of Econmics Policies.
